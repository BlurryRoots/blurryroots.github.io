Working remotely with sublime text.

There are two basic scenarios.

Edit random files

To do that, i'd recommend using rsub. There is a great tutorial by tedbot explaining how to do that.

Working on a remote project

In this case you want to work on a particular part of your file system. In order to archive this, you want to mount the remote directory over ssh. Here you'll find a comprehensive tutorial how to set up sshfs, which enables you to simply do something like:

$ sshfs username@yourhostname.com:/path/to/remote/directory /path/to/local/
